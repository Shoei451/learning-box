# Unit 08 · 食中毒の予防

## 食中毒とは

有害な微生物などが付着した食品を食べることによって起きる健康障害。  
季節を問わず発生するが、高温多湿な夏や冬のウイルス性のものが特に多い。

## 食中毒予防の3原則

| 原則 | 具体例 |
|------|--------|
| 菌を「**つけない**」 | 手洗い・清潔な調理器具・生肉と野菜を分ける |
| 菌を「**増やさない**」 | **10℃以下**で保存・迅速な調理 |
| 菌を「**やっつける（生かさない）**」 | 中心温度**75℃以上・1分間**以上の加熱 |

> **冷凍食材の解凍は室温でしない！**
> 食中毒菌が最もよく増殖する温度帯（10〜40℃）を長く通過するため、冷蔵庫内または電子レンジで解凍する。

## 主な食中毒の原因と特徴

### 細菌性食中毒

| 菌名 | 特徴 | 主な原因食品 |
|------|------|------------|
| **サルモネラ菌** | ペットからの感染も | 鶏卵・食肉 |
| **カンピロバクター** | 少ない菌数でも発症・近年最多 | 鶏肉（特に生食） |
| **病原性大腸菌O-157** | ベロ毒素産生・重症化すると後遺症（HUS） | 牛肉・生野菜 |
| **腸炎ビブリオ** | 好塩菌・夏に多い | 魚介類（刺身） |
| **黄色ブドウ球菌** | 手指に常在・産生する**毒素は熱に強い** | おにぎり・弁当・和菓子 |
| **ボツリヌス菌** | 酸素のない環境で増殖・**非常に強力な神経毒** | 缶詰・真空パック・ハチミツ |

> **黄色ブドウ球菌の注意点**
> 調理者の手に傷がある場合に特に注意。毒素は加熱しても分解されないため、菌をつけないことが最重要。

### ウイルス性・その他

| 原因 | 特徴 | 主な原因食品 |
|------|------|------------|
| **ノロウイルス** | 冬に多い・感染力が非常に強い・大規模集団感染 | 牡蠣など二枚貝 |
| **アニサキス** | 寄生虫・激しい腹痛 | サバ・イカ・サーモンなどの魚介類 |
| **ヒスタミン** | アレルギー様症状・加熱しても分解されない | 赤身魚（サバ・イワシ） |

> **アニサキスの予防**
> 加熱（60℃以上）または冷凍（-20℃以下・24時間以上）で死滅する。目視でも確認可能。

## 食中毒菌の増殖と温度

```
-20℃以下 → 増殖停止（冷凍）
  10℃以下 → 増殖抑制（冷蔵）
10〜40℃  → 急速に増殖（危険ゾーン）
  75℃以上 → 1分間で多くの菌が死滅（加熱殺菌）
```

## HACCP（ハサップ）

食品の製造工程において、危害要因（微生物・化学物質・異物など）を分析・管理し、安全を確保する衛生管理手法。  
**Hazard Analysis and Critical Control Point** の略。  
2021年から日本でも食品関連事業者への導入が義務化。
