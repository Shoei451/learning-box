Cloze Builder

Markdown に ⟦⟦…⟧⟧ を埋め込んで穴埋め教材を作る。

⟦⟦ 0
.md ファイルをドロップして読み込み
📄 MD ファイルをドロップ
穴埋めプレビュー クリックでトグル
⟦⟦⟧⟧

エディタにMDを入力すると穴埋めプレビューが表示されます

⟦⟦ 0 文字 0 行 0

使い方

  1. Markdown を貼り付けるか、.md ファイルをエディタにドロップして読み込む。
  2. 隠したい語句を選択して ⟦⟦ 穴埋め化（または Alt+W）。手書きで ⟦⟦語句⟧⟧ と書いてもよい。
  3. プレビュータブで表示を確認。クリックで個別トグル、全表示/全隠しも可。
  4. ⬇ ダウンロードして notes/{slug}/unitN.md に保存、Netlify にデプロイ。
記号について： ⟦⟦…⟧⟧ は Markdown・Obsidian・HTML のいずれとも衝突しない安全な記号です。 note-app.js はこれを読み取り、クリックトグル型の穴埋めUIとして描画します。
